# **App Name**: QuoteVid

## Core Features:

- Google Sheets Integration: Fetch phrases and categories dynamically from a connected Google Sheet.
- Phrase Categorization: Display phrases organized by category fetched from Google Sheets.
- Video Creation with Phrases: Allow users to select a phrase and overlay it on a video or image.
- Text Customization: Provide options to customize text: font, color, size, alignment, and shadow. Use an AI tool to suggest complimentary fonts and colors.
- Layout Adjustment: Enable users to adjust the aspect ratio (1:1, 9:16, 16:9) and layout of the video.
- Export and Share: Allow saving the generated video to the device and sharing it directly on social media platforms.
- Search and Filter: A search input that allow to search through the quotes and to filter the quotes based on the given categories.

## Style Guidelines:

- Primary color: A vibrant purple (#A06CD5) to evoke creativity and inspiration.
- Background color: A light, desaturated lavender (#F2E7FA) to provide a clean and modern backdrop.
- Accent color: A contrasting teal (#45B8AC) for interactive elements and highlights, drawing attention without being overwhelming.
- Headline font: 'Poppins' (sans-serif) for a geometric and modern feel in titles.
- Body font: 'PT Sans' (sans-serif) which pairs well with 'Poppins,' providing a balanced and readable text for descriptions and options. 
- Use a consistent set of minimalistic icons that clearly represent actions and categories. Icons should be easily recognizable and fit within the modern design.
- A clean and modern design using clear and logical visual hierachy and consistent spacing. The layout should adapt to different screen sizes.
